# Kode Etik Komunitas

Komunitas **Bansos.dev** berkomitmen membangun kontribusi yang aman, ramah, dan profesional.

## Nilai kami

- Saling menghormati pandangan, bahasa, dan cara kerja orang lain.
- Menjaga komunikasi yang jelas, sopan, dan berfokus pada perbaikan.
- Mengutamakan manfaat komunitas: data akurat, mudah dicek, dan ramah pengguna.

## Aturan perilaku

- Tidak boleh menggunakan bahasa yang menghina, merendahkan, atau melecehkan.
- Tidak mempromosikan spam, phishing, atau program menyesatkan.
- Tidak menyebarkan data pribadi/ sensitif tanpa izin.
- Tidak mengirim PR yang mengandung konten ilegal, diskriminatif, atau merusak reputasi proyek.

## Penanganan pelanggaran

Jika kamu menemukan pelanggaran, laporkan via issue di repository.
Tim maintainers berhak menegur, meminta revisi, atau menutup kontribusi sesuai tingkat dampaknya.

Dengan ikut berkontribusi, kamu menyetujui untuk mengikuti pedoman ini.
