# ✅ Promo Domain Rp 1/Tahun dari Sumopod

**Provider:** Sumopod

Sumopod sedang mengadakan promo pendaftaran domain baru hanya dengan harga Rp 1 untuk tahun pertama.

> **Status:** Aktif

⏰ **Berlaku sampai:** 2026-06-30

## Keuntungan

- Harga domain hanya Rp 1 untuk pendaftaran tahun pertama

## Persyaratan

- Buat akun di platform Sumopod
- Pilih menu Domain
- Masukkan nama domain yang diinginkan
- Lanjutkan ke proses checkout

## Tips

Provider mungkin mengalami pembatasan (rate-limit) karena tingginya permintaan, harap coba berkala.
