# ✅ Cleanesia — API Korpus Teks Bahasa Indonesia Gratis

**Provider:** Deflated

API publik gratis untuk mengakses korpus teks Bahasa Indonesia terkurasi dari berbagai domain, cocok untuk riset NLP, fine-tuning model, dan eksperimen data.

> **Status:** Aktif

⏰ **Info Waktu:** Layanan gratis tanpa periode promo terbatas

## Keuntungan

- Akses gratis ke 70.587+ dokumen teks Bahasa Indonesia terkurasi
- Korpus multi-domain: encyclopedia, legal, dan news
- Dataset tersedia terbuka di Hugging Face (ripkiiiii/cleanesia)
- Endpoint publik dapat diakses tanpa API key atau signup

## Persyaratan

- Buka landing page Cleanesia dan pilih endpoint yang dibutuhkan

## Tips

Coba endpoint contoh: https://api.deflated.xyz/api/sample?domain=encyclopedia&n=5
