# ✅ Freebuff — AI Coding Agent & App Builder 100% Gratis

**Provider:** Freebuff

Platform AI coding agent 100% gratis. CLI untuk coding dari terminal (pengganti Claude Code, Codex, Cursor) dan Web untuk build full-stack app dari prompt (pengganti Lovable, Bolt). Tanpa API key, tanpa kartu kredit.

> **Status:** Aktif

## Keuntungan

- Freebuff CLI: coding agent gratis di terminal
- Freebuff Web: AI app builder dari prompt ke deployed app
- Deploy & host app gratis, 230K+ developer aktif

## Persyaratan

- Node.js 18+ untuk CLI
- Akun gratis untuk Web
