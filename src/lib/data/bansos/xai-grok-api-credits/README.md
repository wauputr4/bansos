# ✅ Kredit Gratis Uji Coba xAI Grok API

**Provider:** xAI

xAI menyediakan kredit promosi bagi developer baru yang ingin menguji Grok API. Terdapat juga opsi kredit bulanan tambahan melalui keikutsertaan dalam program berbagi data.

> **Status:** Aktif

⏰ **Info Waktu:** Kredit awal $25 kedaluwarsa 30 hari setelah pendaftaran

## Keuntungan

- Kredit promosi awal sebesar $25 langsung setelah pendaftaran
- Kredit tambahan hingga $150 per bulan dengan mengikuti program data sharing

## Persyaratan

- Buat akun di xAI Console dan hasilkan API key
- Aktifkan opsi berbagi data di pengaturan akun untuk mendapatkan kredit tambahan $150 per bulan
